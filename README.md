# Student Dashboard Project

This is a responsive student dashboard built for a college assignment. It provides a clean, modern interface for a student to view their key academic metrics at a glance.

The initial UI was generated using [v0 by Vercel](https://v0.dev/) and then customized.

## ‚ú® Features

* **Responsive Header:** Displays a welcome message and student profile avatar.
* **Summary Cards:** A 4-card grid showing:
    * Attendance
    * CGPA
    * Credits Earned
    * Upcoming Exams
* **Attendance Chart:** A doughnut chart (powered by `Chart.js`) visualizing attended vs. missed classes.
* **Upcoming Events:** A list of upcoming notices, deadlines, and events.
* **Fully Responsive:** Adapts seamlessly from mobile phones to desktop screens.
